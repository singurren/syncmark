# Result Log Convention / 结果记录约定

Store dated bilingual step records in this directory. Raw experiment metrics belong in task-specific sibling directories under `results/`.

本目录用于存放按日期记录的双语步骤日志。实验原始指标应放在 `results/` 下按任务划分的同级目录中。

Each step log should include:

每一步日志至少应包含：

- Purpose / 目的
- Work Done / 做了什么
- Result / 结果
- Achievement / 达成了什么
- Interpretation / 说明什么
- Next Step / 后续步骤

Suggested filenames:

建议文件名：

- `YYYY-MM-DD_step_01.md`
- `YYYY-MM-DD_step_02.md`

Raw experiment artifacts can be stored in dated subdirectories next to the narrative logs.

实验原始输出可以放在与叙述日志同级的日期子目录中。
